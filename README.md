# Plants Analysis Using Excel

## 📌 Project Overview
This project analyzes a plant catalog dataset using **Microsoft Excel** (dashboard visuals, pivot tables, charts) to uncover pricing patterns, inventory composition, and brand performance.

## 🗂️ Data Source
- **Source:** Kaggle
- **Original dataset:** `Raw Data/plant_url.xlsx`

## 🧹 Data Cleaning
- **Tool:** Power Query Editor (Excel)
- **Cleaned data file:** `Clean Data/plant_url.xlsx`
- Steps included:
  - Removing duplicates and irrelevant columns
  - Normalizing price fields and categorical labels
  - Filtering/standardizing plant categories (Size, Location, Type, Brand)

## 📊 Analysis (Microsoft Excel)
The Excel analysis focuses on several key areas:

1. **Pricing Analysis**
   - Compare **Original Price vs Special Price** for top products
   - Identify deep discounts and consistent value offerings

2. **Inventory Composition**
   - Distribution by **Plant Size** (Small / Medium / Large)
   - Distribution by **Plant Location** (Indoor / Outdoor / Both)
   - Distribution by **Plant Type** (Shrubs, Flowers, etc.)

3. **Brand Performance**
   - Top brands by unit count
   - Market concentration and long-tail brand structure

4. **Key Observations & Recommendations**
   - Opportunity to introduce **Small** plants to diversify inventory
   - Risk from over-reliance on a single vendor (UGADU)
   - Pricing strategy insights (e.g., deep discount on Jade Plant)
